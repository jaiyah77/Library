flick 동작 후 콜백을 호출 하려면 아래와 같이 callback 옵션에 호출할 함수를 넣어준다. 
$('.flick-sample-06').horizontalBox({
    'ui': 'flick',
    'indicator': true,
    'control': true,
    'callback': function () {
        var data = $('.flick-sample-06').data();
        console.log('This is flick count number:',
                data.horizontalBox.detect.flickCount);
    }
});

연결 하고 싶은 곳 네비$('.navi')의 선택자를 아래와 같은 형태로 $('.flick-sample-08').horizontalBox()호출시 옵션으로 지정해준다.

'navi': {
	wrapper: '.navi',
	onTarget: '.navi-item'
}
연결하고 싶은 네비($('.navi'))의 detect값을 이용하기 위해 스크롤을 반드시 먼저 셋팅해준다. 
$('.navi').horizontalBox();

$('.flick-sample-08').horizontalBox({
    'ui': 'flick',
    'indicator': true,
    'control': true,
    'navi': {
        wrapper: '.navi',
        onTarget: '.navi-item'
    },
    'start': 2
});
